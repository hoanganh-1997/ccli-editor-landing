## Product Overview

**Name:** CCLI Editor
**Tagline:** The all-in-one coding environment built around Claude Code CLI.
**One-liner:** CCLI Editor transforms Claude Code CLI into a complete development environment — combining AI, code management, and dev tools in a single lightweight app.

**What it is:**
A lightweight desktop application for developers who use Claude Code CLI daily. It makes Claude the central interface of your workflow and eliminates context switching by bringing your terminal, file tree, git tools, browser preview, code editor, and token tracking into one window.

**What it is NOT:**

- Not a replacement for Claude Code CLI — Claude stays the core
- Not a bloated IDE with AI bolted on
- Not a general-purpose editor for everyone

## Problem

**The core pain:**
Claude Code CLI is powerful but it's just a terminal. Every time Claude changes something, you have to switch to another tool to see what actually happened — open your browser, check git, navigate your file tree, look at diffs. You're constantly context switching, and it kills your flow.

**Specific friction points:**

1. Have to alt-tab to a browser to see UI changes after every prompt
2. Have to run `git diff` / `git status` in a separate window to review changes
3. No visibility into project structure as Claude modifies it
4. No easy way to do quick manual edits without opening a full IDE
5. Token usage is invisible — no way to track daily/weekly spend
6. Conversation history disappears when you close the terminal

---

## Solution & Key Features

### 1. Claude-Centric Workflow

Claude Code CLI is the core of the application. Open the app and start prompting — no complex setup. Every other feature exists to support the Claude workflow, not compete with it.

### 2. Integrated File Tree

Full project structure navigation inside the app. See exactly what Claude is changing in real time without switching to Finder or another terminal. Track AI-generated changes at a glance instead of relying on terminal output alone.

### 3. Built-in Git Management

Common git operations available through a clean, intuitive interface — no need to open a separate terminal or git GUI. Stage, commit, and push code generated by Claude without leaving CCLI Editor.

### 4. Integrated Browser Preview

Built-in browser preview directly inside the editor. See how your web app looks the moment Claude makes a change — no switching to an external browser, no manual refresh workflow.

### 5. Token Usage Tracking

Real-time token usage monitoring. Track remaining tokens on a daily and weekly basis so you always know what you're spending. No surprises at the end of the month.

### 6. Conversation History

Automatically saves your interaction history with Claude. Revisit and review previous prompts and outputs at any time. Never lose context from a previous session.

### 7. Built-in Code Editor

Make quick manual edits without needing AI assistance — ideal for small fixes, typos, or refinements. No need to open VS Code just to change one line.

### 8. Lightweight & High Performance

Built with Rust (Tauri v2) for maximum performance. Minimal memory usage (~50MB RAM), fast startup (<1 second), smooth experience. ~10MB bundle vs ~150-200MB for Electron-based alternatives.

## Tech Stack (for technical audience)

- **Tauri v2 (Rust)** — ~10MB bundle, <1s boot, ~50MB RAM idle
- **Rust backend** — PTY management (portable-pty), git (git2), filesystem watching (notify)
- **React + TypeScript + Tailwind** — frontend UI
- **xterm.js** — terminal emulation
